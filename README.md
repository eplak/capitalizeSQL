# capitalizeSQL
Capitalizes all SQL keywords in an .sql file.


## How to use
1. put upperCaseSQL.py and SQLkeywords.txt in same directory as sql file
2. run upperCaseSQL.py in terminal (no arguments)
3. write only the filename without the ending. (example: table and NOT table.sql (you can also have the path of the filebefore that if you dont want to have the program in the same directory))
